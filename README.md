# Airline Market Analysis

Data analysis project examining airline ticket pricing, passenger demand, and market trends using real-world airline datasets.

## Overview

This project analyzes airline markets including:
- LAX–SEA
- LGA–ORD

The analysis focuses on:
- Passenger volume trends
- Average fare changes
- Airline competition
- Market behavior over time

## Technologies Used

- Python
- pandas
- matplotlib

## Skills Demonstrated

- Data cleaning
- Data visualization
- Trend analysis
- Market interpretation
- Business analytics

## Key Insights

- Passenger demand fluctuated across markets over time
- Fare trends reflected competitive market behavior
- Market conditions impacted pricing and passenger volume
